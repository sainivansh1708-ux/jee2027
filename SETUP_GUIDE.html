📚 JEE 2027 Tracker — Setup Guide

One-time setup (~15 minutes). After this, your tracker lives at a permanent URL, installs as an app, and syncs to Google Sheets automatically.

Part 1 — Host on GitHub Pages (free permanent URL)

1
Go to github.com and create a free account (if you don't have one). Just needs an email.
2
Click the + icon → New repository. Name it exactly: jee2027. Set it to Public. Click Create repository.
3
In the new repo, click uploading an existing file. Upload all 4 files from the zip: index.html, manifest.json, sw.js, icon-192.png, icon-512.png. Click Commit changes.
4
Go to Settings (top of repo) → Pages (left sidebar) → Under Source, select Deploy from a branch → Branch: main, folder: / (root). Click Save.
5
Wait ~1 minute. Refresh the Pages settings. Your URL will appear: https://yourusername.github.io/jee2027. Bookmark this. This is your permanent tracker URL.
✅ At this point your tracker is live at your GitHub Pages URL. You can already use it — data saves locally in your browser. Now set up Google Sheets sync.

Part 2 — Connect Google Sheets (auto-sync)

1
Go to sheets.google.com. Create a new blank spreadsheet. Name it JEE 2027 Tracker.
2
In the spreadsheet, click Extensions in the top menu → Apps Script. A new tab opens with a code editor. Delete everything already in it.
3
Paste this script into the editor (select all → paste):
function doPost(e) { try { var data = JSON.parse(e.postData.contents); var ss = SpreadsheetApp.getActiveSpreadsheet(); var sheet = ss.getSheetByName('Log') || ss.insertSheet('Log'); if (sheet.getLastRow() === 0) { sheet.appendRow([ 'Date','Hours','Mood','Task %','Maths Notes', 'Physics Notes','Chem Notes','Test Name', 'Test Score','Weak Topics','Reflection' ]); } sheet.appendRow([ data.date, data.hours || '', data.mood || '', data.taskPct || 0, data.mathNotes || '', data.phyNotes || '', data.chemNotes || '', data.test ? data.test.name : '', data.test ? data.test.score : '', data.test ? data.test.weak : '', data.reflection || '' ]); return ContentService .createTextOutput(JSON.stringify({status:'ok'})) .setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService .createTextOutput(JSON.stringify({status:'error',msg:err.toString()})) .setMimeType(ContentService.MimeType.JSON); } }
4
Click the 💾 Save icon (or Ctrl+S). Name the project JEE Tracker Sync.
5
Click Deploy (blue button, top right) → New deployment. Click the gear icon ⚙️ next to "Type" → select Web app.
6
Settings: Description = JEE Tracker · Execute as = Me · Who has access = Anyone. Click Deploy.
7
Google will ask you to authorize. Click Authorize access → choose your Google account → click Allow. Then copy the Web app URL that appears (starts with https://script.google.com/macros/s/...).
8
Open your tracker (yourusername.github.io/jee2027) → tap ⚙️ (settings tab) → paste the URL into the box → click Connect. Done!
✅ From now on, every time you hit "Save today's log" in the tracker, a new row is added to your Google Sheet automatically — even from your phone.

Part 3 — Install as an app on Chrome

1
Open your tracker URL in Chrome on your laptop. Look for a ⊕ install icon in the address bar (right side). Click it → Install.
2
The tracker opens as a standalone window — no browser tabs, no distractions. Pin it to your taskbar. It also works offline (data saves locally first, syncs when online).
3
On Android phone: Open the URL in Chrome → tap the ⋮ menu → Add to Home Screen. It'll appear as an app icon on your home screen.
